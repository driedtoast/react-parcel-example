

## Getting Started ##

* Install nave: `npm install -g nave`
* `nave auto`
* Install packages: `npm install` 

To run:
* `npm start` 
* Go to [http://localhost:1234/]


## Libraries Used ##
* (Nave)[https://www.npmjs.com/package/nave]
* (Parcel)[https://parceljs.org/]
* (React)[https://reactjs.org/]
